This directory contains the proxy, a WebSocket server that interfaces the
[backend](/backend) and the web [frontend](/frontend).

## Prerequisities

- [Rust](https://www.rust-lang.org/tools/install) 1.90.0+

## Build

```sh
cargo build
```
